# Campus Club Management System (CCMS)

The Campus Club Management System (CCMS) is a relational SQL-based database designed to manage
student clubs within a university environment. The system supports club administration,
student memberships, event management, and role assignments.

This project demonstrates practical database design skills, data integrity enforcement,
and structured query development using SQL.

## Technologies
- MySQL
- SQL

## Database Features
- Relational database design
- Normalized tables (up to 3NF)
- Primary and foreign key relationships
- Constraints (NOT NULL, UNIQUE, FOREIGN KEY)
- Many-to-many relationships using junction tables
- Complex SELECT queries with JOIN operations

## Core Entities
- Students
- Clubs
- Memberships
- Events
- Roles

## Functionality
- Register students and clubs
- Assign students to clubs with specific roles
- Manage club events
- Track memberships and participation
- Enforce data consistency and integrity

## Purpose
Academic project developed as part of an Applied Computing programme, demonstrating
database modelling, SQL querying, and relational database concepts applicable to real-world systems.
